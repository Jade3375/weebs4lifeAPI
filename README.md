# weebs4lifeAPI
api wrapper for weebs4life.ga
```bash
npm i weebs4life
```
how to use
```js
const weeb = require('weebs4life');
const weebs = new weeb()

weebs.sfw("slap").then(img => {
    console.log(img)
})
```

endpoints
- slap
- kiss
- hug
- neko

for an up to date list on endpoints visit [weebs4life](https://weebs4life.ga/api)
